# Як писати гарний код швидко

Щоб писати гарний код швидко, потрібно:

* **Регулярно практикуватись**, розвʼязуючи задачі на *LeetCode* і *Codewars*;
* **Читати і аналізувати** код інших після виконання задачі;
* **Писати чистий і зрозумілий код** та коментувати складні моменти;
* **Використовувати Git** для контролю версій та пушити код на *GitHub*;
* **Самостійно відлагоджувати** код з дебагером.
